# Changelog

All notable changes to the **ViserFly** project will be documented in this file.

## [1.0.0] - 2025-12-28
### Added
- Initial release of ViserFly 3D Flight Tracker.
- Implemented MVC folder structure for security.
- Added 3D Plane model using Three.js.
- Integrated real-time stats simulation (Speed, Altitude, Pitch).
- Added .htaccess for URL rewriting and security protection.
- Responsive UI with Glassmorphism design.

### Security
- Protected core system files (`app/`) from direct public access.
